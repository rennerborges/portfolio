Sobre mim

Olá, tudo bem? Meu nome é Renner sou um entusiasta por tecnologia, estudante e desenvolvedor Full-Stack. Fui estudante do Curso Técnico em Informática durante o período de 2017 a 2019 no Instituto Federal Goiano Campus Morrinhos e atualmente estou no Curso de Tecnólogo em Sistemas para Internet no mesmo campus. Desde pequeno tenho familiaridade com o computador, amo desenvolver e adoro um desafio, prezo sempre pelo melhor resultado e por um trabalho coletivo.

Conhecimentos

HTML

HTML é uma linguagem de marcação utizada na construção de página web

CSS

CSS é um mecanismo para adicionar estilo a um documento web

JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.

ReactJS

O React é uma biblioteca JavaScript de código aberto com foco em criar interfaces de usuário em páginas web.

Node.js

Node.js trata-se de um software open-source, cross-platform, e de um runtime de JavaScript que execute código de JavaScript a nível backend e frontend.

Sequelize

Sequelize é um Node.js ORM baseado em promessa para Postgres, MySQL, MariaDB, SQLite e Microsoft SQL Server.

Materialize

Materialize é um frameprojeto responsivo baseado no Material Design que foi utilizado pela Google

PHP

PHP é uma linguagem interpretada livre, usada originalmente apenas para o desenvolvimento de aplicações presentes e atuantes no lado do servidor, capazes de gerar conteúdo dinâmico na World Wide Web.
